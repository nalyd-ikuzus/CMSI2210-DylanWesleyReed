Definitions for the following terms:

1. What is a computer system?
- A computer system is a combination of hardware parts that come together to accomplish given tasks, like computations, displaying videos/pictures, or running operating systems

2. What are some of the parts of a computer system?
- Some of the parts of a computer system include the central processing unit, memory, long term storage, motherboard, and possibly video graphics cards and cooling solutions

3. What is the difference between a compiled language and an intepreted language?
- A compiled language is translated by the compiler into machine code before execution, while an interpreted language is interpreted into machine code during execution.

4. Is C a compiled language?
- Yes, C is a compiled language

5. Who invented the C language?
- The C language was invented by Dennis M. Ritchie in the early 1970s.

6. How long has C been in use?
- C has been in use for about 50 years.

7. Is a compiler a translator?
- Yes, a compiler translates high-level/human-readable code into assembly

8. Is an assembler a translator?
- Yes, an assembler translates assembly code into machine code

9. What is the command to list out the contents of a directory on a mac terminal window?
- On a mac terminal window, the command to list the content of a directory is 'ls'.

10. What does the C function atof() do?
- The atof() function takes an array of characters and turns it into a float

11. What are the bottom two layers of a computer system? Give a brief desciption of each.
- Layer 0 is digital logic, which consists of circuits, gates, and other hardware-based digital logic.
- Layer 1 is the control layer, which consists of microcode, AKA firmware, that controls the deeply embedded systems

12. What are the three steps of the Von Neumann Architecture?
- 1. Fetch
- 2. Decode
- 3. Execute

13. What is the purpose of an ALU?
- One of the important things that the ALU, or Arithmetic Logic Unit, does is decode the instructions given to it using the instruction set architecture

14. What is a register?
- A register is ultra-fast, volatile, short-term memory that is usually extremely close to the CPU.

15. What is one difference between Application software and System software?
- Application software is generally dependent on human input/interaction, while System software is more behind-the-scenes/works internally with the computer itself.

16. Is the phrase cmp rdi, rsi machine language?
- No, the phrase cmp rdi, rsi is assembly, which is a level above machine code.

17. How many buses are included in the system bus?
- There are three buses on the system bus:
    1. The data bus
    2. The address bus
    3. The control bus

18. What is the decimal value of 100101112?
- The decimal value of 10010111(2) is 151(10)

19. What is the decimal value of 111111112?
- The decimal value of 11111111(2) is (2^8)-1, which is 255

20. What is the largest unsigned integer value that will fit into 16 bits?
- The largest 16-bit unsigned integer value is (2^16)-1, which is 65535(10)
